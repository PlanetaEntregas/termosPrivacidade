Modo e processamento local de dados

Método de processamento

O Proprietário toma as medidas de segurança permitidas para impedir o acesso não autorizado, divulgação, alteração ou destruição não autorizada dos dados. O processamento dos dados é realizado com a utilização de computadores e / ou ferramentas de TI habilitadas, seguindo procedimentos organizacionais e métodos estritamente relacionados com os fins indicados. Além do proprietário, em alguns casos, os dados podem ser acessados por certos tipos de pessoas envolvidas, usando a operação deste Serviço (este aplicativo) (administração, vendas, marketing, administração legal do sistema) ou pessoas externas (como fornecedores terceirizados de serviços técnicos, carteiros, provedores de hospedagem, empresas de TI, agências de comunicação), quando necessário, como Processadores de dados por parte do proprietário. Uma lista dessas partes pode ser solicitada ao Proprietário a qualquer momento.

Base jurídica para processamento

O Proprietário pode processar os Dados Pessoais relacionados ao Usuário se uma das hipóteses seguir após aplicar:

  • Os usuários têm dado sua solicitação para uma ou mais finalidades específicas; Observação: acordo com algumas legislações ou proprietário pode ter permissão para processar dados pessoais até que o usuário pode fazer objeção a isto (“optar por não participar”), sem ter que se basear em anuência ou em outras bases jurídicas a seguir. Isso não aplica nem sempre o processamento de dados pessoais sujeitos a legislação europeia de proteção de dados;
  • o fornecimento dos dados para o cumprimento de um contrato com o usuário e / ou requisitos mínimos pré-contratuais do mesmo;
  • o processamento para o cumprimento de uma obrigação jurídica para o qual o proprietário está sujeito;
  • o processamento está relacionado a uma tarefa que é executada no interesse público ou no exercício de uma autorização oficial qualificada ou o proprietário está investido;
  • o processamento para requisitos de interesse legítimos perseguidos pelo Proprietário ou por um terceiro;

Em qualquer caso, ou Proprietário colaborador de bom desempenho para esclarecer qual é a base jurídica que se aplica ao processamento, e em especial se for fornecido dados de um requisito obrigatório por força de lei ou contratual, ou uma utilização abusiva para celebrar um contrato.

Lugar

Os dados são processados nas operações de propriedade dos proprietários, e em outros locais onde as peças usadas com o processamento localizam-se. Dependendo da localização do Usuário, como transferências de dados podem envolver a transferência de dados do Usuário para outro país que não seja o seu. Para descobrir mais sobre o processamento local de dados transferidos pelos Usuários podem verificar a seção incluindo os detalhes sobre o processamento de Dados Pessoais. Os Usuários também possuem o direito de serem informados sobre a base jurídica de dados de transferência para países da União Europeia ou para listas de direitos internacionais regidos pelo direito internacional público ou formado por dois ou mais países, como ONU, e sobre como medidas de segurança tomadas pelo proprietário para proteger seus dados.

Se ocorrerem tais alterações, os usuários poderão descobrir mais a respeito de verificar como os requisitos deste documento ou a solicitação ao proprietário usando como informações fornecidas na seção de contatos.

Período de conservação

Os Dados Pessoais serão processados e armazenados pelo tempo necessário para as finalidades das principais coletadas.

Portanto:

  • Os Dados Pessoais coletados para finalidades relacionadas com a execução de um contrato entre o Proprietário e o Usuário ficarão conservados até que o contrato tenha sido completamente cumprido.
  • Os Dados Pessoais coletados para finalidades relacionadas com os legítimos interesses do Proprietário serão conservados pelo tempo que for necessário para cumprir tais finalidades. Os Usuários podem obter informações específicas sobre os interesses legítimos perseguidos pelo Proprietário dentro das restrições relacionadas a este documento ou entrar em contato com o Proprietário.

O Proprietário pode ter permissão de conservação dos Dados Pessoais por um período maior, sempre que o Usuário tiver dado sua permissão para processamento, enquanto que a autorização não tiver sido retirada. Além disso, o Proprietário pode permanecer obrigado a conservar os Dados Pessoais por um prazo maior em todas as ocasiões em que estiver obrigado a aplicá-lo para cumprir uma obrigação jurídica ou cumprir um mandato de uma autoridade.

Assim que o prazo de conservação vence, os dados pessoais serão apagados. Desta forma, o direito de acesso, o direito de excluir, o direito de corrigir e o direito de portabilidade dos dados não podem ter o seu cumprimento exigido após o vencimento do prazo de conservação.

As finalidades do processamento

Os dados relacionados ao usuário são coletados para permitir que o proprietário preste seu serviço, cumpra suas obrigações legais, responda a solicitações de execução, proteja seus direitos e interesses (ou aqueles de seus usuários ou terceiros), detecte qualquer atividade maliciosa ou fraudulenta, assim como: Testes Beta. Para obter informações específicas sobre os dados pessoais usados para cada uso, o usuário pode consultar a seção "Informações sobre o processamento de dados pessoais".

Informações sobre o processamento de dados pessoais

Os Dados Pessoais são coletados para os seguintes fins e usando os seguintes serviços:

Os direitos dos usuários

Os usuários podem exercer direitos autorais e respeitar seus dados processados pelo proprietário. Em especial, os Usuários possuem os direitos autorais a seguir:

  • Retire sua anuência a qualquer momento. Os Usuários possuem o direito de retirar a sua solicitação nos casos em que tenham recebido a sua solicitação anteriormente para o processamento dos seus Dados Pessoais.
  • Objetar o processamento dos seus dados. Os Usuários possuem o direito de objetar ou processar seus dados. Se o processamento de executar sobre outra base jurídica que não for anuência. São selecionados detalhes adicionais na seção específica abaixo.
  • Acessar os seus Dados. Os usuários possuem o direito de saber se os seus dados estão sendo processados pelo proprietário, obter revelações sobre os aspectos do processamento e obter uma cópia dos dados que estão sendo processados.
  • Verificar e pedir retificação. Os usuários possuem o direito de verificar a exatidão dos seus dados e solicitar que os mesmos sejam removidos ou corrigidos.
  • Restringir o processamento dos seus dados. Os Usuários possuem o direito de, sob circunstâncias, restringir o processamento dos seus dados para qualquer outro uso que não seja o armazenamento dos mesmos.
  • Ter os seus dados pessoais apagados ou retirados de outra maneira. Os usuários possuem o direito de, sob circunstâncias mínimas, obter a alteração dos seus dados do proprietário.
  • Receba os seus dados e os mesmos transferidos para outro driver. Os usuários possuem o direito de receber seus dados em um formato estruturado, usado comum e adequado para ser lido por máquinas e, se for viável tecnicamente, fazer com que os mesmos sejam transmitidos para outro driver sem empecilho. Isso determina se os dispositivos processados são processados por meios automatizados e que podem ser usados com base em anuência do Usuário, em um contrato do qualificador ou no Usuário, que é uma das partes ou por direitos prévios do mesmo.
  • Registrar uma reclamação. Os usuários possuem o direito de apresentar reclamação perante sua autoridade de proteção de dados competentes.

Detalhes sobre o direito de objetar ao processamento

Nos casos em que os Dados Pessoais são processados por um interesse público, nenhum exercício de uma autorização oficial para quem é proprietário ou investido ou para finalidades de interesses legítimos perseguidos por proprietário, os usuários usam objetos como o uso de um motivo relacionado com a sua situação em especial para justificar uma objeção.

Nos casos em que os Dados Pessoais são processados por um interesse público, nenhum exercício de uma autorização oficial para quem é proprietário ou investido ou para finalidades de interesses legítimos perseguidos por proprietário, os usuários usam objetos como o uso de um motivo relacionado com a sua situação em especial para justificar uma objeção.

Como exercer estes direitos

Quaisquer pedidos para exercer os direitos dos usuários podem ser direcionados ao Proprietário através dos dados para contato neste documento. Esses pedidos podem ser exercitados sem nenhum custo e serem atendidos pelo Proprietário com uma maior brevidade possível e em todos os casos com prazo inferior a um mês.
